# Cursor Rules — Marketing / Investor Landings

## Tone
- Calm, confident, investor-grade
- Never salesy, never hype-driven

## Content
- Semantic HTML
- Clear headings (H1 → H2 → H3)
- Text-first, document-like layouts

## SEO & AI
- Metadata API
- schema.org where relevant
- MDX or typed data for content

## UI
- shadcn/ui primitives only
- No SaaS-looking components
- Subtle CTAs (text links preferred)
- **Hover States**: Always ensure interactive elements (buttons, links) have clearly defined hover states, appropriate cursors (`cursor-pointer`), and smooth transitions.

## shadcn studio
- Exploration only
- Never part of final stack
